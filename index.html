Fire Alarm System Using IC555

Description:

The Fire Alarm System is a simple and cost-effective solution designed to detect fire and alert users. Utilizing the IC555 timer, the system is capable of sensing smoke fill up in the room and triggering an alarm when a threshold is exceeded.

Goals:

  • To develop a reliable and affordable fire alarm system.
  • To understand and implement the IC555 timer in a practical application.
  • To enhance safety measures by providing an early warning system for fire hazards.

Technologies Used:

  • IC555 Timer: The core component for the timing and detection circuit.
  • Thermistor: A resistor used to detect smoke in the room.
  • Buzzer: An audio alert component to sound the alarm.
  • Resistors and Capacitors: For circuit stability and proper functioning.
  • Power Supply: To power the circuit components.

Features:

  • Smoke Detection: Senses smoke using a thermistor.
  • Audible Alarm: Emits a loud sound to alert users of potential fire.
  • Compact Design: Easy to install and maintain.
  • Low Cost: Economical components ensure affordability.

Challenges:

  • Sensitivity Calibration: Adjusting the thermistor and IC555 circuit to accurately detect significant smoke without false alarms.
  • Power Management: Ensuring the circuit operates efficiently with minimal power consumption.
  • Component Sourcing: Finding high-quality and reliable components within budget constraints.

Outcome:

The project successfully produced a functioning fire alarm system that effectively detects smoke and sounds an alarm. It proved to be a valuable learning experience in circuit design and practical applications of the IC555 timer.

Home Audio System Project

Description:

An integrated audio platform combining a graphic equalizer, class-D amplifier, and OLED spectrogram display. The system processes audio input, enhances user control over sound frequencies, and visualizes real-time signal levels.

Goals:

  • To develop an end-to-end audio processing system with adjustable frequency bands and high-efficiency amplification.
  • To integrate real-time visualization via OLED for enhanced user experience.
  • To apply analog/digital circuit design, signal processing, and microcontroller programming in one unified platform.

Technologies Used:

  • Graphic Equalizer: 2-stage RC bandpass filters for bass (60–500 Hz), mid (600 Hz–3 kHz), and treble (3–6 kHz).
  • Class-D Amplifier: PWM modulator with triangle and comparator circuit; RLC low-pass output filter.
  • OLED Display: Arduino-driven spectrogram displaying intensity of each frequency band in real-time.
  • LTspice: Simulation of filters, amplifier stages, and waveform validation.
  • Analog Discovery 2: Hardware-level frequency and distortion testing.

Features:

  • Adjustable gain control for bass, mid, and treble using potentiometers.
  • High-efficiency class-D amplification using NMOS/PMOS switching and RLC filtering.
  • OLED spectrogram updates every 200 ms, reflecting live frequency intensity data.
  • Intuitive hardware and UI design including welcome screen and dynamic bar graph rendering.

Challenges:

  • Signal noise from Arduino mitigated via averaging and thresholding techniques.
  • Initial PWM switching overlap fixed by adding dead-time between transistor transitions.
  • Ripple voltage at amplifier output reduced through RLC component recalibration.

Outcome:

The system was successfully validated via oscilloscope and spectrum analysis tools, showing low harmonic distortion and accurate band separation. The project demonstrated efficient subsystem integration, engineering design maturity, and user-centric audio visualization. It provides a solid base for future extensions including Bluetooth streaming and advanced frequency analysis.
